## 项目概述

Mighty Coupons 是一个功能完整的全栈优惠券管理平台，我在项目中担任**组长**及**大模型助手开发者**。系统集成了 AI 驱动的智能优惠券生成、商户管理、联盟营销推广等功能。

## 我的角色与贡献

- **组长**：负责项目整体规划、任务分配、代码审查与团队协作
- **大模型助手开发者**：使用 LangChain 编排模型，调用 OpenAI API 构建智能优惠券生成助手
- **实时通信**：利用 WebSocket 实现模型与后端服务器的实时通信，处理异步通信，提升系统响应速度

## 核心功能

- **AI 智能优惠券助手**：基于 LangChain + OpenAI 的对话式优惠券创建，自动生成定制化优惠方案
- **优惠券管理系统**：完整的优惠券 CRUD 操作、状态流转、数据统计
- **联盟营销计划**：支持推广链接生成、佣金计算、推广效果追踪
- **海报编辑器**：Vue.js 实现的可视化海报设计工具

## 技术栈

`Python` `LangChain` `OpenAI API` `PostgreSQL` `HTMX` `Bootstrap 5` `Vue.js` `WebSocket`
