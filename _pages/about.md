# 你好，我是武盛玮 👋

我是上海第二工业大学**网络工程**专业的本科生，同时也是一名热爱技术的开发者与安全工程师。

## 🔐 安全领域

在[绿盟科技](https://www.nsfocus.com.cn/)实习期间，我独立完成了 **20+ 套**政府、金融、运营商系统的渗透测试，累计发现 **90+ 个有效漏洞**，并参与了国家级攻防演练。这段经历让我对 Web 安全、API 安全和系统防御有了深入的理解。

## 🤖 AI 与大模型应用

我对大模型（LLM）应用开发充满热情。目前在**优刻得（UCloud）**从事 AI Infra 方向，专注于大模型推理加速和 GraphRAG 等 Agent 应用开发。此前在多个个人项目中实践了 RAG（检索增强生成）、LangChain 编排、vLLM 推理部署等技术：

- **RAG 文档检索系统** — 基于 LlamaIndex + Qdrant 的文档问答服务
- **量化交易平台 SinoQuant** — 集成多 LLM 提供商的智能股票分析系统
- **医疗语音 AI 平台 MedVoice** — 基于 vLLM 的医疗语音交互系统

## 💻 全栈开发

我具备 Python、C++、JavaScript 的全栈开发能力，熟悉 FastAPI、Vue.js 等框架。在 RAG 优惠券系统中担任组长，使用 WebSocket 实现实时通信；在公平抽签系统中应用非对称加密算法实现无需信任第三方的随机数生成。
