# 欢迎来到我的技术博客 🎉

这是我在 GitHub Pages 上搭建的个人技术博客。在这里，我将分享我在安全工程、AI 应用开发、全栈开发等领域的学习笔记和实践经验。

## 我是谁

我是武盛玮，上海第二工业大学网络工程专业的本科生。目前是一名安全服务工程师，同时也是 Python 和 C++ 开发者。

## 我的技术方向

我的技术兴趣主要集中在三个领域：

### 🔐 网络安全

在绿盟科技的实习经历让我深入了解了渗透测试、漏洞挖掘和攻防演练。我独立完成了 20+ 套系统的渗透测试，发现了 90+ 个有效漏洞。

### 🤖 AI 与大模型应用

从 vLLM 推理部署到 RAG 系统开发，再到基于 LangChain 的智能助手，我在多个项目中积累了丰富的 LLM 应用开发经验。

### 💻 全栈开发

FastAPI、Vue.js、PostgreSQL —— 我享受从零开始构建完整应用的过程。
